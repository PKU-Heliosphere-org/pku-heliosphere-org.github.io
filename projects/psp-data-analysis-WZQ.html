项目概览

PSP Data Analysis 是一个专门用于处理和分析帕克太阳探针（Parker Solar Probe, PSP）观测数据的Python工具包。 该项目旨在为太阳物理研究人员提供一套完整的数据处理、分析和可视化工具，帮助研究人员更好地理解太阳风和日冕加热等物理过程。

核心优势

  • 支持多种PSP仪器数据的读取和处理
  • 自动化的数据下载和更新机制
  • 交互式数据可视化
  • 完整的轨道遭遇数据分析流程

功能特性

数据获取与处理

  • 自动化数据下载
  • 数据格式转换
  • 数据质量控制
  • 时间序列处理

数据分析工具

  • 等离子体参数计算
  • 磁场分析
  • 波动分析
  • 统计分析

可视化功能

  • 交互式时序图
  • 轨道可视化
  • 参数关联分析
  • 自定义绘图模板

使用示例

基本数据处理


import psp_tools as psp

# 下载特定时间段的PSP数据
data = psp.download_data('2022-01-01', '2022-01-02')

# 读取并预处理数据
processed_data = psp.process_data(data)

# 计算等离子体参数
params = psp.calculate_plasma_params(processed_data)

数据可视化


# 创建交互式时序图
fig = psp.plot_timeseries(params, 
                         variables=['density', 'temperature', 'velocity'],
                         interactive=True)

# 保存为HTML文件
psp.save_plot(fig, 'plasma_parameters.html')
