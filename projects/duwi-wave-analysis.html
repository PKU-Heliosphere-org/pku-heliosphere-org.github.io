项目概览

DUWI Wave Analysis 是一个专门用于分析太阳风和行星际空间中等离子体波动的工具包。 该项目提供了一系列用于处理和分析空间物理波动数据的Python工具，帮助研究人员更好地理解太阳风和行星际空间中的等离子体波动现象。

主要功能包括：

  • 太阳风和行星际空间等离子体数据的读取与预处理
  • 等离子体波动特征的识别与分析
  • 频谱分析和时频分析
  • 波动模式分类和特征提取

核心功能

数据处理

支持多种格式的DUWI数据导入和预处理，包括数据清理和校准。

波动分析

提供多种波动分析工具，包括频谱分析、小波分析等。

可视化

强大的可视化功能，支持时间序列、频谱图和波动特征的可视化。

统计分析

包含波动特征的统计分析工具，支持多维数据分析。

使用示例

以下是一些常见使用场景的代码示例：

基本数据处理


import duwi_tools as dt

# 加载太阳风等离子体波动数据
data = dt.load_data('solar_wind_data.cdf')

# 数据预处理和校准
processed_data = dt.preprocess(data, calibrate=True)

波动分析


# 进行频谱分析和小波变换
spectrum = dt.spectrum_analysis(processed_data)
wavelet = dt.wavelet_transform(processed_data)

# 绘制频谱图和小波图
dt.plot_spectrum(spectrum)
dt.plot_wavelet(wavelet, time_range=['2023-01-01', '2023-01-02'])
